Portfólio - Vitória Pinheiro

Este é o portfólio pessoal de Vitória Pinheiro, desenvolvido em React com TypeScript e Vite.
O projeto apresenta informações pessoais, habilidades e exemplos de projetos realizados, além de um formulário de contato.

Tecnologias utilizadas

React

TypeScript

Vite

CSS

Funcionalidades

Menu interativo que alterna entre as seções Sobre, Projetos e Contato.

Perfil clicável que mostra informações adicionais.

Formulário de contato simples.

Layout responsivo e limpo.
